Installation

You can install VelocityFX using npm or include it directly via CDN.

Using NPM

npm install velocityfx

Using CDN

<!-- Add to your <head> tag -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/velocityfx@latest/dist/velocityfx.min.css">
<script src="https://cdn.jsdelivr.net/npm/velocityfx@latest/dist/velocityfx.min.js"></script>

Basic Usage

VelocityFX uses HTML attributes to define animations. Here are some basic examples:

Entry Animations

<div vfx-animate="fade-in">
    This will fade in when visible
</div>

Hover Animations

<button vfx-hover="scale-up">
    Hover me!
</button>

Configuration

You can configure VelocityFX by passing options when initializing:

const vfx = new VelocityFX({
    threshold: 0.1,
    rootMargin: '20px',
    duration: 300,
    easing: 'ease',
    debug: false
});

Animations

VelocityFX provides a rich set of animations for different interactions:

Entry Animations

  • fade-in - Fade in from transparent to opaque
  • slide-up - Slide up from bottom
  • slide-down - Slide down from top
  • slide-left - Slide in from left
  • slide-right - Slide in from right
  • zoom-in - Scale up from smaller size
  • zoom-out - Scale down from larger size
  • flip - Flip animation on the Y-axis
  • flip-x - Flip animation on the X-axis
  • rotate - Rotate into view

Hover Animations

  • scale-up - Scale up on hover
  • scale-down - Scale down on hover
  • glow - Add a glowing effect
  • shake - Shake animation
  • pulse - Pulsing animation
  • wobble - Wobble effect
  • float - Floating effect
  • bounce - Bouncing effect

Click Animations

  • pop - Pop effect on click
  • bounce - Bounce effect on click
  • ripple - Ripple effect from click point
  • flash - Quick flash effect

Transitions

Customize transition properties for animations:

// Using data attributes
<div vfx-animate="fade-in"
     vfx-duration="500"
     vfx-easing="ease-out"
     vfx-delay="200">
    Content
</div>

// Using JavaScript
const vfx = new VelocityFX({
    duration: 500,
    easing: 'ease-out',
    delay: 200
});

Sequences

Create staggered animations with sequences:

<div vfx-sequence vfx-sequence-delay="200">
    <div vfx-animate="fade-in">First</div>
    <div vfx-animate="fade-in">Second</div>
    <div vfx-animate="fade-in">Third</div>
</div>

Sequence Options

  • vfx-sequence-delay - Delay between each animation (ms)
  • vfx-sequence-direction - Animation direction (forward/reverse)
  • vfx-sequence-stagger - Custom stagger pattern

Parallax Effects

Add depth to your website with parallax scrolling:

<div vfx-parallax="0.5">
    Moves at half speed
</div>

<div vfx-parallax="1.5">
    Moves faster than scroll
</div>

Parallax Options

  • vfx-parallax - Scroll speed multiplier
  • vfx-parallax-direction - Scroll direction (vertical/horizontal)
  • vfx-parallax-perspective - 3D perspective value

Accessibility

VelocityFX is built with accessibility in mind:

Reduced Motion

// Automatically respects prefers-reduced-motion
@media (prefers-reduced-motion: reduce) {
    * {
        animation: none !important;
        transition: none !important;
    }
}

ARIA Attributes

<div vfx-animate="fade-in"
     aria-hidden="true"
     role="presentation">
    Decorative animation
</div>

Performance Optimization

Best practices for optimal performance:

  • Use will-change for hardware acceleration
  • Implement throttling for scroll-based animations
  • Lazy load animations for better initial page load
  • Use vfx-threshold to control animation timing
// Example of performance optimization
const vfx = new VelocityFX({
    threshold: 0.1,
    throttle: 16, // ~60fps
    useWillChange: true,
    lazyLoad: true
});
