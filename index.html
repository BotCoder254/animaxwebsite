Beautiful Animations Made Simple

VelocityFX is a lightweight and powerful animation framework that makes it easy to add beautiful animations to your web applications.

Get Started View Examples 
<!-- Add beautiful animations with simple HTML attributes -->
<div vfx-animate="fade-in">
    <h1>Welcome to my website</h1>
</div>

<button vfx-hover="scale-up">
    Hover me!
</button>

<div vfx-sequence vfx-sequence-delay="200">
    <div vfx-animate="slide-up">First</div>
    <div vfx-animate="slide-up">Second</div>
    <div vfx-animate="slide-up">Third</div>
</div>
