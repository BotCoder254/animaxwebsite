API Reference

Initialization

Initialize VelocityFX with custom options:

// Initialize with default options
const vfx = new VelocityFX();

// Initialize with custom options
const vfx = new VelocityFX({
    threshold: 0.1,
    rootMargin: '20px',
    duration: 300,
    easing: 'ease',
    debug: false
});

Methods

Available methods for controlling animations:

animate(element, options)

Manually trigger an animation on an element.

vfx.animate(element, {
    animation: 'fade-in',
    duration: 300,
    easing: 'ease-out'
});

observe(element)

Start observing an element for visibility changes.

vfx.observe(element);

destroy()

Clean up all observers and event listeners.

vfx.destroy();

Entry Animations

Available entry animations and their parameters:

Fade Animations

  • fade-in - Fade from transparent to opaque
  • fade-out - Fade from opaque to transparent
vfx.animate(element, {
    animation: 'fade-in',
    opacity: [0, 1],  // [start, end]
    duration: 300
});

Slide Animations

  • slide-up - Slide up from bottom
  • slide-down - Slide down from top
  • slide-left - Slide in from left
  • slide-right - Slide in from right
vfx.animate(element, {
    animation: 'slide-up',
    distance: '100px',  // Distance to slide
    duration: 500
});

Scale and Rotate

  • zoom-in - Scale up from smaller size
  • zoom-out - Scale down from larger size
  • flip - Flip on Y-axis
  • flip-x - Flip on X-axis
  • rotate - Rotate into view
vfx.animate(element, {
    animation: 'flip',
    degrees: 180,
    duration: 600
});

Hover Effects

Available hover animations and their parameters:

Scale Effects

  • scale-up - Scale up on hover
  • scale-down - Scale down on hover
vfx.hover(element, {
    animation: 'scale-up',
    scale: 1.1,  // Target scale
    duration: 200
});

Visual Effects

  • glow - Glowing effect
  • shake - Shake animation
  • pulse - Pulsing effect
  • wobble - Wobble animation
  • float - Floating effect
  • bounce - Bouncing effect
vfx.hover(element, {
    animation: 'glow',
    intensity: 0.5,  // Effect intensity
    duration: 300
});

Click Animations

Available click animations and their parameters:

Click Effects

  • pop - Pop effect on click
  • bounce - Bounce effect on click
  • ripple - Ripple effect from click point
  • flash - Quick flash effect
vfx.click(element, {
    animation: 'ripple',
    color: '#6366f1',  // Ripple color
    duration: 400
});

Sequence Animations

Control and customize sequence animations:

Sequence Configuration

vfx.sequence(container, {
    delay: 200,  // Delay between items
    direction: 'forward',  // or 'reverse'
    stagger: 'linear',  // or 'random'
    children: '.item'  // Selector for sequence items
});

Events

Available events and their payloads:

Animation Events

  • vfx:animate:start - Animation started
  • vfx:animate:end - Animation completed
  • vfx:animate:cancel - Animation cancelled
document.addEventListener('vfx:animate:start', (event) => {
    const { element, animation, options } = event.detail;
    console.log(`${animation} started on`, element);
});

Interaction Events

  • vfx:hover:start - Hover started
  • vfx:hover:end - Hover ended
  • vfx:click - Click animation triggered
element.addEventListener('vfx:hover:start', (event) => {
    const { effect, options } = event.detail;
    console.log(`Hover effect ${effect} started`);
});

Sequence Events

  • vfx:sequence:start - Sequence started
  • vfx:sequence:progress - Sequence progress
  • vfx:sequence:end - Sequence completed
container.addEventListener('vfx:sequence:progress', (event) => {
    const { current, total, progress } = event.detail;
    console.log(`Sequence progress: ${progress}%`);
});

Observer Events

  • vfx:visible - Element entered viewport
  • vfx:hidden - Element left viewport
  • vfx:parallax - Parallax scroll update
element.addEventListener('vfx:visible', (event) => {
    const { ratio } = event.detail;
    console.log(`Element visibility: ${ratio}`);
});

Callbacks

Available callback functions for animations:

Animation Callbacks

vfx.animate(element, {
    animation: 'fade-in',
    onStart: (element) => {
        console.log('Animation started');
    },
    onProgress: (element, progress) => {
        console.log(`Progress: ${progress}%`);
    },
    onComplete: (element) => {
        console.log('Animation completed');
    },
    onCancel: (element) => {
        console.log('Animation cancelled');
    }
});
